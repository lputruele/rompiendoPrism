This case study concerns robots moving concurrently over a grid.

It models the system in PRISM-games as a concurrent stochastic game (CSG) [KNPS19].

For more information, see: http://www.prismmodelchecker.org/casestudies/robot_coordination.php

=====================================================================================

Equilibria-based Probabilistic Model Checking for Concurrent Stochastic Games
